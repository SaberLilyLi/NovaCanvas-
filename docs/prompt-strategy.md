# Prompt Strategy

## DeepSeek

DeepSeek 只承担任务规划，不直接面向前端输出说明。系统 Prompt 强制返回 JSON，包含：

- 任务类型
- 图片数量
- 并行或串行模式
- 是否继承历史图片
- 每个子任务的完整 Prompt、参考图与尺寸

服务端会再次通过 `buildImagePrompt` 叠加业务约束，防止规划结果绕过行业质量规则。

## 二手车

重点保持车辆品牌特征、车身比例、轮毂、车灯、玻璃与漆面真实性。背景替换不能改变车辆
身份，营销图强调销售可信度，不生成虚假车牌或乱码文字。

## 服装

重点保持版型、领型、袖型、面料与装饰细节。多图参考允许一张提供款式、一张提供风格，
Prompt 强调穿搭逻辑、材质触感、人物姿态与编辑摄影质感。

## 历史图片继承

本轮没有上传图片，且指令包含“这张、上一张、继续、把它、基于它、再调整”时，规划器将
`latestImageId` 作为输入图。用户明确上传新图时，优先使用本轮图片。
